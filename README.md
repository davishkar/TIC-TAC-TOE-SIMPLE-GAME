# Tic Tac Toe Game

## Overview

This project is a simple implementation of the classic Tic Tac Toe game using HTML, CSS, and JavaScript. It provides a two-player experience, allowing players to take turns and notifies them of the game's outcome.

## Features

- **Player Display**: Indicates which player's turn it is.
- **Interactive Board**: Clickable tiles for players to make their moves.
- **Announcement**: Displays the winner or a tie message when the game concludes.
- **Reset Button**: Allows users to reset the game and start over.

## Technologies Used

- **HTML**: Markup language for structuring the web page.
- **CSS**: Stylesheet language for enhancing the visual presentation.
- **JavaScript**: Programming language for interactive game logic.

## How to Play

1. Open the [Tic Tac Toe](#) webpage in a browser.
2. Players take turns clicking on the empty tiles to make their moves.
3. The game announces the winner or declares a tie when the game concludes.
4. Use the "Reset" button to start a new game.

## Game Logic

- The game checks for winning conditions after each move.
- If a player wins or the game ends in a tie, an announcement is displayed.
- The game can be reset at any time to start a new round.
